# MEttLE
Temporary Working Repository of MEttLE during the summer of 2017

# Usage
 - install node (latest)
 - install mongodb(use default port)
 - clone this repository `git clone https://github.com/rohinrohin/MEttLE`
 - `cd MEttLE` 
 - `npm install`
 
 To run the server
  - `npm run debug`
